Sitio web estático con Express.js

Este proyecto utiliza Express.js para servir un sitio web estático basado en una plantilla Bootstrap.

Rutas disponibles
- / → Página principal del sitio

- Cómo ejecutar el proyecto
1. Instalar dependencias
   npm install

2. Ejecutar el servidor
   node index.js

3. Abrir en el navegador
   http://localhost:3000
